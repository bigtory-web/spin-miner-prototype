# spin-miner-prototype

브레이크 시스템과 채굴 루프 검증을 중심으로 정리한 스핀 마이너 프로젝트 저장소입니다.

## Structure

- `docs/`: 시스템/맥락 문서
- `prototype/`: 실행용 프로토타입
- `prompts/`: 프로젝트 전용 프롬프트
- `assets/`: 포트폴리오 자산
- `archive/`: 이전 산출물, 임시/보관 자료

## Note

이 저장소는 기존 원격 이력을 유지하면서, 현재 작업 기준을 `docs / prototype / assets / archive` 구조로 정리한 버전입니다.
