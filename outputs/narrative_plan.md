# Spin Miner Portfolio Draft Narrative Plan

## Audience
Supercent 콘텐츠/시스템 기획 포지션 리뷰어.

## Objective
원작을 수정했다는 표현보다, 라이브 게임 관찰을 바탕으로 대형 광물 구간의 조작 목적과 보상감을 강화하는 시스템 제안과 프로토타입 검증 과정을 보여준다.

## Narrative Arc
1. 원작 루프 관찰
2. 대형 광물 구간의 경험 문제 정의
3. 약점과 BREAK 시스템 가설
4. 플레이 가능한 V1 프로토타입 범위
5. 규칙, 보상 루프, 밸런스 기준 정리
6. 테스트 중 조정한 항목
7. 실제 원작 시점 적용 시 V2 고려사항

## Slide List
1. Cover
2. Project Positioning
3. Original Loop Observation
4. Problem Definition
5. Hypothesis
6. Prototype Scope
7. System Rules
8. Reward Loop
9. Balance Design
10. Iteration Log
11. V2 Direction
12. Summary

## Visual Plan
각 슬라이드는 추후 사용자가 직접 게임 캡처를 교체할 수 있도록 큰 이미지 자리와 간단한 캡션을 둔다. 핵심 이미지는 원작 400B 대형 광물, 프로토타입 약점 노출, BREAK 연출, 광석 드랍/수집 장면이다.

## Editability Plan
슬라이드의 제목, 본문, 표, 플로우, 이미지 자리는 모두 PowerPoint에서 직접 수정 가능한 텍스트/도형으로 구성한다.
